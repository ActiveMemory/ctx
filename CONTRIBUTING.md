## Contributing to Context

Thank you for your interest in contributing to `ctx`.

**Full contributing guide**: https://ctx.ist/contributing/

The guide covers:

- Development setup (binary + plugin from your local clone)
- Project layout and the skills directory structure
- Day-to-day workflow (Go changes, skill changes, tests)
- Submitting pull requests
- Legal (DCO, license, code of conduct)
